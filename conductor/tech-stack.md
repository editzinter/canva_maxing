# Technology Stack

## Core Framework & Language
- **Framework:** [Next.js](https://nextjs.org/) (App Router, v16.1.0)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (v5)
- **Runtime:** [Node.js](https://nodejs.org/)

## Frontend & UI
- **Library:** [React](https://reactjs.org/) (v19.2.3)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4) with [PostCSS](https://postcss.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (v12.23.26)
- **Icons:** [Lucide React](https://lucide.dev/) (v0.562.0)
- **Theme Management:** [Next-themes](https://github.com/pacocoursey/next-themes) (v0.4.6)
- **Smooth Scrolling:** [Lenis](https://lenis.darkroom.engineering/) (v1.3.16)

## Graphics & Canvas
- **Canvas Engine:** [Fabric.js](http://fabricjs.com/) (v6.5.4)
- **Image Export:** [html-to-image](https://github.com/bubkoo/html-to-image) (v1.11.13)
- **PDF Generation:** [jsPDF](https://github.com/parallax/jsPDF) (v3.0.4)

## Development & Tooling
- **Linting:** [ESLint](https://eslint.org/) (v9)
- **Type Checking:** TypeScript Compiler
